## Installation

<Tabs>
  <Tab title="Desktop Apps">
    Add to your MCP client config (Claude Desktop, ChatGPT, Gemini, etc.):

    ```json
    {
      "mcpServers": {
        "rewind": {
          "command": "npx",
          "args": ["-y", "rewind-mcp-server"],
          "env": {
            "REWIND_API_KEY": "rw_live_your_key_here"
          }
        }
      }
    }
    ```

    Restart the app after saving.

  </Tab>
  <Tab title="Mobile & Web">
    Add as a remote integration in your client's settings:

    - **URL**: `https://mcp.rewind.rest/mcp`
    - **Authorization**: `Bearer rw_live_your_key_here`

    Works with any MCP client that supports remote servers.

  </Tab>
  <Tab title="Claude Code">
    ```bash
    claude mcp add rewind -- npx -y rewind-mcp-server
    ```

    Or add `.mcp.json` to your project root:

    ```json
    {
      "mcpServers": {
        "rewind": {
          "command": "npx",
          "args": ["-y", "rewind-mcp-server"],
          "env": {
            "REWIND_API_KEY": "rw_live_your_key_here"
          }
        }
      }
    }
    ```

  </Tab>
</Tabs>

Requires a [Rewind API key](/authentication). `REWIND_API_URL` defaults to `https://api.rewind.rest`.

## Available Tools

### Listening

| Tool                    | Description                                                  |
| ----------------------- | ------------------------------------------------------------ |
| `get_now_playing`       | Currently playing track                                      |
| `get_recent_listens`    | Recent scrobbles with date filtering                         |
| `get_listening_stats`   | Total scrobbles, unique artists/albums/tracks, daily average |
| `get_top_artists`       | Top artists by period (7d, 1m, 3m, 6m, 12m, all-time)        |
| `get_top_albums`        | Top albums by period                                         |
| `get_top_tracks`        | Top tracks by period                                         |
| `get_listening_streaks` | Current and longest consecutive listening streaks            |
| `get_artist_details`    | Artist info, play count, top albums and tracks               |
| `get_album_details`     | Album info, play count, track listing                        |

### Running

| Tool                   | Description                                                     |
| ---------------------- | --------------------------------------------------------------- |
| `get_running_stats`    | Total runs, distance, elevation, pace, Eddington number         |
| `get_recent_runs`      | Recent activities with distance, pace, duration, location       |
| `get_personal_records` | Fastest times at mile, 5K, 10K, half marathon, marathon         |
| `get_running_streaks`  | Current and longest consecutive running streaks                 |
| `get_activity_details` | Full details for a single run (heart rate, elevation, calories) |
| `get_activity_splits`  | Per-mile splits with pace, elevation, and heart rate            |

### Watching

| Tool                 | Description                                             |
| -------------------- | ------------------------------------------------------- |
| `get_recent_watches` | Recently watched movies and TV shows                    |
| `get_movie_details`  | Director, genres, rating, summary, watch history        |
| `browse_movies`      | Browse by genre, decade, director, or year with sorting |
| `get_watching_stats` | Total movies, watch time, top genre/director, TV stats  |

### Collecting

| Tool                       | Description                                                   |
| -------------------------- | ------------------------------------------------------------- |
| `get_vinyl_collection`     | Browse vinyl records with search, format/genre/artist filters |
| `get_collecting_stats`     | Total items, format breakdown, unique artists, genre data     |
| `get_physical_media`       | Browse Blu-ray, 4K UHD, and HD DVD collection                 |
| `get_physical_media_stats` | Physical media totals and format breakdown                    |

### Reading

| Tool                     | Description                                                      |
| ------------------------ | ---------------------------------------------------------------- |
| `get_recent_reads`       | Recently saved articles with title, author, domain, reading time |
| `get_reading_highlights` | Saved highlights with text, notes, and source article            |
| `get_random_highlight`   | A single random highlight for inspiration or reflection          |
| `get_reading_stats`      | Total articles, finished count, highlights, word count           |

### Cross-Domain

| Tool              | Description                                            |
| ----------------- | ------------------------------------------------------ |
| `search`          | Full-text search across all domains (or filter to one) |
| `get_feed`        | Unified chronological feed of all activity             |
| `get_on_this_day` | What happened on this date in previous years           |

## Prompt Templates

The server includes built-in prompt templates for common multi-step queries:

| Prompt            | Description                                              |
| ----------------- | -------------------------------------------------------- |
| `weekly-summary`  | Summarize activity across all domains for the past week  |
| `year-in-review`  | Generate a comprehensive year-in-review for a given year |
| `compare-periods` | Compare activity between two time periods for a domain   |

## Environment Variables

- **`REWIND_API_KEY`** (required) -- Your Rewind API key
- **`REWIND_API_URL`** (optional) -- API base URL, defaults to `https://api.rewind.rest`

## Security

<Info>
  The MCP server has **read-only access** to your data. No write or admin
  operations are exposed.
</Info>

- **Local mode**: Your API key stays on your machine in the local config file. The server runs as a child process and communicates over stdin/stdout.
- **Remote mode**: Your API key is sent to `mcp.rewind.rest`, a Cloudflare Worker you control. All communication is over HTTPS.
- The server is stateless -- no data is cached or stored locally.

## What is MCP?

[Model Context Protocol](https://modelcontextprotocol.io/) (MCP) is an open standard created by Anthropic that allows AI applications to connect to external tools and data sources. Any MCP-compatible client -- Claude, Cursor, VS Code, Windsurf, and others -- can use the Rewind MCP server to access your data directly during a conversation.
