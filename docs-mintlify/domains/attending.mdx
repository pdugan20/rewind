## Overview

The Attending domain tracks live events you bought tickets for: sports games, concerts, theater, and other one-off events. Events come from Google Calendar (when Google's parsers populate ticketing details) and Gmail (per-vendor parsers for Ticketmaster, SeatGeek, Ticket Club, AXS, StubHub, VividSeats, Eventbrite, plus a universal `schema.org/EventReservation` JSON-LD parser).

Sports games are enriched with team and score data from the MLB Stats API and ESPN (NFL, NBA, WNBA, MLS, NCAAF, NCAAB). Concerts can cross-link to Last.fm artists for unified listening + attendance views. Data syncs daily at 4:00 AM Pacific via cron with incremental Gmail and Calendar pulls.

## Key endpoints

| Endpoint                                                                               | Description                                                  |

| [GET /attending/seasons/:league/:season](/api-reference/attending/get-attended-season) | Attended games in a sports season with W/L record            |

## Event categories

Every event has a top-level `category` and a more specific `event_type`:

| Category | Event types                                                                             |
| -------- | --------------------------------------------------------------------------------------- |
| `sports` | `mlb_game`, `nfl_game`, `nba_game`, `wnba_game`, `mls_game`, `ncaaf_game`, `ncaab_game` |
| `music`  | `concert`                                                                               |
| `arts`   | `theater`, `comedy`, `other`                                                            |

Sports `event_type` determines which league API enriches the row. `event_data` carries league-specific fields — for MLB games that's `home_team`, `away_team`, `home_score`, `away_score`, `winning_pitcher`, `my_team_won`.

## Attendance vs ownership

The `attended` flag separates "I went" from "I bought tickets but didn't go." Both states are returned by default — pass `attended=1` to filter to confirmed attendance, `attended=0` to see no-shows. The seasons endpoint only counts wins and losses for events you actually attended, so a 6-game home season with one no-show reports a 5-1 attended record rather than the full 5-2 team record.
